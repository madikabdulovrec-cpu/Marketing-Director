## 1. Контекст

**Заказчик:** Дмитрий Дишинди — владелец и пар-мастер Dishandi SPA, авторская методика парения **Bardo**.

**Локация:** Алматы, Коктобе 18.

**Продукт:** премиум-баня с авторским ритуалом парения Bardo. Без алкоголя, халяль-формат, ритуал перехода в другое состояние длительностью до 6 часов.

**Целевая аудитория:**
- A. Бизнес-мужчины 35-50 (ядро, 75%)
- B. ЗОЖ + духовные практики (растущий сегмент)
- C. Женские группы
- D. Корпоративные клиенты (B2B)
- E. Спортсмены / биохакинг
- F. Иностранцы / экспаты

**Анти-сегмент:** компании на застолье / алкоголь.

## 2. Бизнес-проблема

```
2025 (легкий режим)         →  2026 (масштаб)
──────────────────────         ────────────────
Выручка:  33,0 млн ₸          Выручка:  31,4 млн ₸ за 5 мес (×2,3)
Прибыль:  17,1 млн ₸          Прибыль:   8,8 млн ₸ за 5 мес (+22%)
Маржа:    52%                  Маржа:    28% средняя

Помесячная маржа 2026: 30% (ноя) → 28% (дек) → 34% (янв) → 25% (фев) → 18% (март)
                                                            ↑ обвал
```

**Ключевая проблема:** маржа сжимается. Фиксированные расходы выросли в 4-5 раз быстрее, чем выручка. Главный фактор — сезонность, требующая стабилизации.

**Воронка:**
- 2GIS — 4,7/5 рейтинг, но без коллтрекинга → слепая зона
- WhatsApp — 60 000 ₸/мес на таргет → 45 заявок → 2-3 бани (конверсия 6%, норма 15-25%) → главная дыра
- Сарафан — основной канал, без CRM
- Партнёрской сети — нет

## 5. Ключевые цифры и решения

### Юнит-экономика
- Цена: 50 000 ₸ / человек (минимум 3 чел)
- Contribution margin: ~67% (стабильна при любом N)
- Break-even: **14 сеансов/мес** (при N=5,7)
- Capacity max: 60 сеансов/мес

### Цели на 90 дней
| Метрика | База (март 26) | 90 дней |
|---|---|---|
| Сеансов / мес | 17 | **32** |
| Загрузка | 28% | **53%** |
| Выручка | 5,2 млн | **9,4 млн** |
| Маржа | 18% | **35%** |
| Конверсия WhatsApp | 6% | **15%** |
| Партнёров активных | 0 | **6** |
| База в Altegio | 0 | **300+** |

### Совокупный эффект 4 тактик (к 90-му дню)
| Инициатива | Бюджет | Доп. contribution/мес |
|---|---|---|
| 8.1 Invitation Club | 240к | +100к |
| 8.2 Арома-якорение | 200к | +200к (retention) |
| 8.3 Bardo Trio | 60к | +860к |
| 8.4 B2B | 150к | +550к |
| **Итого** | **650к** | **+1,71 млн** |
