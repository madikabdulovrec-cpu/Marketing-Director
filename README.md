# Marketing-Director — рабочий репозиторий маркетинг-директора

Мульти-проектный репозиторий: стратегии, контент-планы, прототипы, документация.

## С чего начать (особенно на новом устройстве)

1. **Прочитай [`memory/MEMORY.md`](memory/MEMORY.md)** — это индекс всех проектов с их статусом, локацией кода и готовыми промптами для запуска сессии Claude.
2. **Если работаешь в Claude Code:** [`CLAUDE.md`](CLAUDE.md) уже автоматически направляет ассистента в `memory/`.
3. Открой нужную карточку проекта в [`memory/project_*.md`](memory/) и работай по ней.

## Что в Dishandi-стратегии (v1.3)

- Краткое резюме и три ставки на 90 дней
- Ситуационный анализ + финансовая модель (P&L, юнит-экономика, BE)
- SWOT, STP (сегменты, JTBD, CJM, VPC), маркетинг-микс 7P + IMC
- SMART-цели и стратегические приоритеты
- Тактические инициативы: Invitation Club, Арома-якорение, Bardo Trio, B2B
- Roadmap 30/60/90, KPI, Retention Ladder, риски
