# T.S — приложение для сети АЗС

Прототипы мобильного приложения для сети АЗС **T.S** (Казахстан, 16 точек).

## Запуск сессии по этому проекту

> Работаем над T.S app. Прочитай `memory/project_ts_app.md`. Прототипы в `ТС приложение/TS_app_design_v*.html` и `TS_app_prototype*.html`. Жду задачу.

## Где что лежит (в git)

| Файл | Что |
|---|---|
| `ТС приложение/TS_app_design_v1..v4.html` | Итеративные версии дизайн-прототипа (v4 — последняя итерация дизайна) |
| `ТС приложение/TS_app_prototype.html` | Финальный кликабельный прототип |
| `ТС приложение/TS_app_prototype_bright.html` | Яркая версия финального прототипа |
| `ТС приложение/_extracted/tz_*.png` | PNG-фрагменты ТЗ (страница, четверти) |

## Что НЕ в git (только локально)

| Файл / папка | Почему |
|---|---|
| `ТС приложение/T.S Brandbook.pdf` | Брендбук заказчика, ~32 MB |
| `ТС приложение/_extracted/brandbook/` | 78 MB PNG-извлечений брендбука |
| `ТС приложение/ТЗ для приложения T.S` | PDF ТЗ от заказчика (без расширения) |

Если на новом устройстве не хватает — копируй вручную из старого.

## Контекст

- Сеть из **16 АЗС** (та же сеть, что и в проекте Receipts Bot — это смежные проекты для одного владельца).
- Дизайн опирается на брендбук T.S и ТЗ заказчика.
- HTML-прототипы — самостоятельные файлы, можно открывать прямо в браузере.

## Связь с другими проектами

**Receipts Bot** ([project_receipts_bot.md](project_receipts_bot.md)) — тоже про сеть T.S (16 АЗС). Бот собирает чеки от кассиров, приложение — клиентское. Возможна интеграция в будущем (программа лояльности, привязка чеков к клиенту через QR).
